## 3. 미구현 상세 스펙

### 체결강도 추이 미니차트
```
- 갱신 시마다 체결강도를 히스토리에 누적
- 웹: st.line_chart로 시계열 표시
- 저장: session_state (페이지 새로고침 시 초기화) 또는 파일 저장
```
- **예상 공수**: 1일

### 실시간 체결 내역
```
- 최근 체결 20건: 체결가 / 체결량 / 체결 시각 / 매수·매도 구분
- API: inquire-ccnl (FHKST01010300) 상세 데이터 활용
- 또는: WebSocket H0STCNT0 (3종목 제한 주의)
```
- **예상 공수**: 1일

### 외국인/기관 순매수
```
- 당일 외국인/기관/개인 순매수 수량
- API: FHKST01010900 (종목별 투자자)
- REST rate limit 고려 (종목당 추가 1건 호출)
```
- **예상 공수**: 0.5일

### 개인 매매 설정
```
- 매입가 / 보유 수량 입력 (웹: st.number_input, 사이드바)
- 자동 계산: 평가금액, 평가손익, 수익률, 손익분기점
- 종목별 저장: portfolio.json
```
- **예상 공수**: 1일

### 목표가 / 손절가
```
- 목표가 → 목표 수익금 자동 계산
- 손절가 → 최대 손실금 자동 계산
- 호가창에 해당 가격대 하이라이트
```
- **예상 공수**: 1일

### 알림 기능
```
- 지정가 도달 시 st.toast 알림
- 체결강도 급변 (30% 이상 변화) 알림
- VI 근접 (3% 이내) 알림
- Streamlit 자동 갱신 (st.rerun + 타이머) 필요
```
- **예상 공수**: 1일

### Claude AI 분석기
```
- ClaudeAnalyzerAdapter 추가 (AnalyzerPort 구현)
- 호가창 + 시세 데이터를 프롬프트로 전달
- 설정/사이드바에서 분석기 전환 (규칙 기반 ↔ AI)
- ANTHROPIC_API_KEY 필요 (.env)
```
- **예상 공수**: 1일

## 5. KIS API 연동 현황

### REST API

| tr_id | API | 설명 | 상태 |
|---|---|---|---|
| FHKST01010100 | inquire-price | 현재가, 등락, 거래량, 52주, VI, 고저가 | ✅ 연동 완료 |
| FHKST01010300 | inquire-ccnl | 체결강도, 체결 내역 | ✅ 연동 (체결강도 사용, 체결 내역 미사용) |
| CTPF1002R | search-info | 종목명 조회 | ✅ 연동 완료 |
| FHKST01010400 | inquire-daily-price | 일별 시세 (이동평균 계산) | ✅ 연동 완료 |
| FHKST01010900 | inquire-investor | 종목별 투자자 (외국인/기관) | ❌ 미연동 |

### WebSocket

| tr_id | 설명 | 상태 | 비고 |
|---|---|---|---|
| H0STASP0 | 실시간 호가 (10단계) | ✅ 연동 완료 | 3종목 제한, 배치 교대 |
| H0STCNT0 | 실시간 체결 | ❌ 미연동 | 3종목 제한에 포함 |
| H0STVI0 | VI 발동 | ❌ 미연동 | 3종목 제한에 포함 |
